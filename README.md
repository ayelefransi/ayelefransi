## ⚙️ Setup Guide for Advanced Widgets

### 3D Contribution Graph

1. Create a new repo named `ayelefransi/ayelefransi` (your GitHub profile repo)
2. Add this GitHub Action file at `.github/workflows/profile-3d.yml`:

```yaml
name: GitHub-Profile-3D-Contrib

on:
  schedule:
    - cron: "0 18 * * *"
  workflow_dispatch:
  push:
    branches:
      - main

jobs:
  build:
    runs-on: ubuntu-latest
    name: generate-github-profile-3d-contrib
    steps:
      - uses: actions/checkout@v3
      - uses: yoshi389111/github-profile-3d-contrib@0.7.1
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
          USERNAME: ayelefransi
      - uses: stefanzweifel/git-auto-commit-action@v4
        with:
          commit_message: generate 3d contrib
```

### WakaTime Setup

1. Sign up at [wakatime.com](https://wakatime.com)
2. Install the WakaTime plugin in VS Code or your IDE
3. Add your WakaTime API key as a GitHub secret named `WAKATIME_API_KEY`
4. Your username in the stats card above must match your WakaTime username
