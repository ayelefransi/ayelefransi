<details>
<summary>⚙️ Setup: 3D Contribution Graph</summary>

Create `.github/workflows/profile-3d.yml` in your profile repo:

```yaml
name: GitHub-Profile-3D-Contrib

on:
  schedule:
    - cron: "0 18 * * *"
  workflow_dispatch:
  push:
    branches:
      - main

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: yoshi389111/github-profile-3d-contrib@0.7.1
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
          USERNAME: ayelefransi
      - uses: stefanzweifel/git-auto-commit-action@v4
        with:
          commit_message: generate 3d contrib
```

It runs daily at 18:00 UTC and self-commits the SVG output.
